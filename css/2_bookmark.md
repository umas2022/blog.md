# 开源库收集

- 鬼畜震动
  - [CSShake](http://elrumordelaluz.github.io/csshake/)
  - pm i csshake -s
  - 使用时候在style里import
```css
@import '~csshake';
div.shake-box {
    display: inline-block;
    margin: auto;
    width: 300px;
    height: 200px;
    background-color: red;
}
```
